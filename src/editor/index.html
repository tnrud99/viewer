How to Use Simple Editor

🎯 Simple Timestamp Editor

Load your timestamp file and edit timestamps by dragging blocks on the timeline.

Quick Guide:

  1. Load File: Click "Choose File" to load your timestamp JSON
  2. Edit Timestamps: Drag blocks to adjust timing
  3. Add/Delete: Use buttons to add new timestamps or delete selected ones
  4. Export: Download the edited timestamps as JSON

Keyboard Shortcuts:

  • Delete: Delete selected timestamp
  • Ctrl+A: Add new timestamp
  • Escape: Deselect timestamp
